# Whammy DT веб контроллер

Веб-инструмент для упралвения Digitech Whammy DT с помощью MIDI прямо из браузера.

## Онлайн версия
https://stevegrey7.github.io/whammydt-web-controller/

## Возможности
- Переключение режимов блоков Whammy и Drop
- Настройка до 8 хоткеев

## Как использовать
- Открой ссылку выше (используйте браузер с поддержкой Web MIDI API, например Chrome).
- Разрешите сайту доступ к вашим MIDI устройствам если требуется.
- В MIDI Device выберите устройство с которого будет отправляться миди сигнал на вашу педаль WhammyDT.
- В Channel выберите миди канал соответствующий тому, что выставлен в вашем WhammyDT (либо оставьте Omni).
- Теперь вы можете переключать режимы нажимая на кнопки включения в блоках Harmony/Whammy и Drop.
   
## Как настроить Hotkeys
- В блоке Hotkeys нажмите на кнопку справа с надписью EDIT.
- Далее выберите слот который хотите отредактировать нажав на него, его контур окрасится в жёлтый.
- Когда слот выбран, нажмите на нужный режим в блоках Harmony/Whammy или Drop который хотите привязать к этому слоту.
- После этого внутри слота в поле введите желаемую горячую клавишу на клавиатуре.
- Когда закончите редактировать - нажмите кнопку EDIT еще раз. Теперь на нажатие заданных клавиш будут переключаться соответствующие режимы.
